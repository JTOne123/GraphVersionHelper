# GraphVersionHelper

A light helper to determine which are currently versions of Facebook Graph API for HTTP request without update your applications every time a new version is available.


# Supported platforms
- .NET Framework 4.5;
- Silverlight 5;
- Windows 8;
- Windows Phone 8.1;
- Windows Phone Silverlight 8;
- Xamarin.Android;
- Xamarin.iOS;
- Xamarin.iOS (Classic).


# Dependencies
- Microsoft.Net.Http;
- Microsoft.Bcl;
- Microsoft.Bcl.Build;
- Microsoft.Bcl.Async.
